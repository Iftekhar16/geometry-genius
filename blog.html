Q&A
What is DOM and what is the purpose of DOM?
DOM is known as Document Object Model.
It is mainly used to get elements from a document and make objects out of them, thus making them interactive.
How to select elements from DOM?
There are multiple ways to select elements from DOM. But you need to have a way to call them.
For example, if you have an element with an ID, you can use
document.getElementById("your-id");
Similarly, if you have one element or multiple elemnets with the same class, you should use
document.getElementsByClass("your-class");
You can also get all the same tags in the following way
document.getElementsByTagName("your-tag");
What is event bubble?
The name event bubble comes from the concept of a bubble going up if its underwater.
How javascript works if you call an event in an innermost element, it will keep calling the upper elements that are wrapping the inner element.
This can be handy for complex websites where there are a lot of events everywhere.
But if you want to stop this calling chai, you can use
.stopPropagation();
What is callback function and how is it used?
Callback functions are basically functions passed in as a parameter of another function.
This can be useful to control the flow or to write cleaner code.
